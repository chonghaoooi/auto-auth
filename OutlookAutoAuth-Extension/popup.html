How to Find Your Secret Key

When setting up Microsoft Authenticator, choose "I want to use a different authenticator app". A QR code will appear — click "Can't scan image?" to reveal the secret key (a string of letters & numbers). Copy that key here.

If already set up, ask IT to reset your MFA so you can re-enroll and capture the key.
